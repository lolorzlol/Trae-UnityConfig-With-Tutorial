# 文档编写规范

## 基本原则

- 使用简洁的中文表达
- 代码示例必须可运行
- 添加必要的图示说明
- 保持格式统一

## 文档结构

每个文档应包含：

1. **标题**：清晰描述文档主题
2. **概述**：简要说明文档目的和适用范围
3. **正文**：分章节详细说明
4. **示例**：提供可运行的代码示例
5. **参考**：相关链接和扩展阅读

## 格式要求

### 标题层级

```markdown
# 一级标题（文档标题）
## 二级标题（章节）
### 三级标题（小节）
```

### 代码块

使用语法高亮：

````markdown
```csharp
public void ExampleMethod()
{
    Debug.Log("Hello World");
}
```
````

### 列表

- 无序列表使用 `-`
- 有序列表使用 `1. `、`2. `

### 表格

```markdown
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
```

## 常见问题

- 避免过长的段落，每段不超过 5 行
- 避免过度嵌套，最多 3 层
- 重要内容使用 **加粗** 强调
