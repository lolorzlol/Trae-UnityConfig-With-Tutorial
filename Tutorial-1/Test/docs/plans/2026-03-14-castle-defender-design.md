# 《城堡守护者》游戏设计文档

**版本**: 1.0  
**日期**: 2026-03-14  
**状态**: 已确认

---

## 1. 项目概述

### 游戏名称
《城堡守护者》(Castle Defender)

### 游戏类型
3D 第三人称 中世纪基地防御生存游戏

### 核心概念
一款3D第三人称的中世纪基地防御生存游戏。玩家拥有自己的城堡据点，日间收集资源、建造城墙与防御设施，夜间敌人会波次进攻。玩家可以自由切换近战（剑/斧）和远程（弓/弩）武器，在城堡内移动战斗，抵御敌人入侵并保护核心城堡。

### 游戏时长
短篇体验，约30分钟-1小时通关

---

## 2. 核心机制

### 2.1 核心循环

```
收集资源 → 建造/升级防御 → 击退敌人波次 → 获得奖励 → 继续发展
```

### 2.2 战斗系统

| 类别 | 武器 | 特点 |
|------|------|------|
| 近战 | 剑 | 攻速快，范围攻击 |
| 近战 | 斧 | 伤害高，单体输出 |
| 远程 | 弓 | 射速快，单发伤害低 |
| 远程 | 弩 | 射速慢，伤害高，可穿透 |

- 玩家可自由切换近战/远程武器
- 武器通过击退敌人获得经验升级

### 2.3 建造系统

- **模式**: 固定基地 - 围绕核心城堡扩建
- **建造方式**: 在指定区域放置建筑
- **升级系统**: 建筑可消耗资源升级

### 2.4 资源系统

| 资源 | 用途 | 获取方式 |
|------|------|----------|
| 木材 | 建造初级建筑 | 砍树 |
| 石头 | 建造石制建筑 | 采矿 |
| 食物 | 维持体力 | 打猎/采集 |
| 金属 | 制作高级武器装备 | 采矿 |

## 3. 敌人设计

### 3.1 敌人类型

| 敌人 | 生命值 | 攻击力 | 特点 | 应对方式 |
|------|--------|--------|------|----------|
| 步兵 | 低 | 低 | 数量多，成群进攻 | 近战武器快速清理 |
| 弓箭手 | 中 | 中 | 远程攻击，会优先射击玩家 | 优先消灭或找掩体 |
| 投石车 | 低 | 高 | 远程重伤害，对建筑威胁大 | 快速摧毁 |
| 狼 | 低 | 中 | 速度快，围攻 | 保持距离 |
| 怪物(Boss) | 高 | 高 | 血厚攻高，小弟多 | 连续攻击+防御设施协助 |

### 3.2 敌人波次

- **触发**: 夜间
- **间隔**: 每波间隔5-10分钟准备时间
- **波次数量**: 5-8波
- **难度递增**: 每波敌人数量和强度逐渐增加

---

## 4. 建筑设计

### 4.1 建筑类型

| 类别 | 建筑 | 功能 | 升级 |
|------|------|------|------|
| 核心 | 城堡之心 | 核心目标，被破坏则游戏结束 | 增加血量上限 |
| 防御 | 城墙 | 阻挡敌人前进 | 血量+耐久 |
| 防御 | 箭塔 | 自动射击范围内敌人 | 射速+伤害 |
| 防御 | 大门 | 控制进出，可开关 | 血量+防御 |
| 防御 | 拒马 | 减速敌人 | 范围+减速效果 |
| 防御 | 陷阱 | 一次性造成大量伤害 | 伤害+触发范围 |
| 辅助 | 仓库 | 增加资源上限 | 容量+ |
| 辅助 | 民居 | 自动产出少量资源 | 产量+ |

### 4.2 建造规则

- 建筑必须在基地范围内建造
- 部分建筑需要前置建筑解锁
- 升级需要消耗对应资源

## 5. 难度设计（心流框架）

### 5.1 心流曲线

按照游戏设计原理，采用心流框架设计难度曲线：

| 阶段 | 波次 | 敌人 | 设计意图 |
|------|------|------|----------|
| 入门 | 1-2 | 少量步兵 | 教学基本操作和建造 |
| 练习 | 3-4 | 步兵+少量弓箭手 | 引入远程威胁 |
| 挑战 | 5-6 | 混合部队+投石车 | 考验多线作战能力 |
| 高潮 | 7-8 | Boss+大量杂兵 | 终极挑战 |

### 5.2 动态难度调整

- 监控玩家连续失败次数
- 触发阈值：连续3次失败
- 调整手段：减少下一波敌人数量10%

---

## 6. 关卡流程

### 6.1 游戏阶段

```
开始 → 白天(准备) → 夜间(防御) → 白天 → ... → 胜利/失败
```

### 6.2 胜利条件
- 成功击退所有敌人波次（8波）

### 6.3 失败条件
- 城堡之心被破坏

---

## 7. 视觉风格

### 7.1 美术风格
- **整体**: 低多边形（Low Poly）中世纪风格
- **色调**: 温暖的棕色调 + 绿色草地
- **比例**: 略微夸张的卡通比例

### 7.2 场景氛围
- **白天**: 明亮温暖，阳光明媚
- **夜间**: 暗色调，火把照明，紧张感
- **UI**: 简洁的中世纪风格，木质边框

## 8. 交互设计

### 8.1 操作方式

| 操作 | 功能 |
|------|------|
| WASD | 移动 |
| 鼠标 | 瞄准 |
| 左键 | 攻击 |
| 1/2 | 切换近战/远程武器 |
| E | 建造/交互 |
| ESC | 暂停菜单 |

### 8.2 UI元素

- 资源栏（顶部）
- 武器切换（底部中央）
- 建造菜单（按E触发）
- 波次提示（屏幕中央）
- 生命值（玩家血条+城堡血条）
